# Kruidenschans 24, Voorhout — projectdossier

Dossier voor de verbouwstudie: **achtergevel optrekken (nokverhoging-variant met
gelijke nok) + 2 dakkapellen + dak opnieuw isoleren**.

## Snelle samenvatting
- Woning **type M4**, plan Oosthout B1, bouwjaar 1986; **hoekwoning**, perceel ± 215 m².
- Fundering op **heipalen**, **breedplaatvloeren**, steil zadeldak (goot +3,70 / nok +9,20).
- Plan: nok **blijft gelijk** (+9,20), **achtergevel + achter-goot optrekken**,
  achterdakhelling steiler → **± +15 à +22 m²** volwaardige bruikbare ruimte.
- **Past binnen het omgevingsplan** (goot ≤ 6 m, nok ≤ 11 m) — door gebruiker
  bevestigd via "Regels op de kaart".
- Nog te doen: constructeursberekening + omgevingsvergunning (bouw + welstand).

> aanvraag tegen de originele gemaatvoerde tekeningen leggen en door een
